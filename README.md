This project is a **full-featured Todo application** built with **React**.
The application allows users to **create, edit, delete, and manage tasks**, as well as filter them by status and title for better organization.

The project demonstrates **component-based architecture, state management**, and **interaction with an external API**, including loading states, error handling, and optimistic UI updates.
It also showcases real-world React patterns such as controlled forms, conditional rendering, and modular code structure.

**Built with:**
- HTML5 for semantic structure
- SCSS for modular, maintainable styles
- BEM methodology for clean, structured code
- Responsive design that adapts beautifully to all screen sizes
- React
- TypeScript
- React Router

**Key Features:**
- Multi-page application with client-side routing
- Product catalog with categorized electronics
- Product details pages
- Responsive navigation menu
- Reusable React components
- Clean and scalable project structure
- Fully responsive layout for mobile, tablet, and desktop



**Getting Started:**
- Clone the repository:
  - git clone https://github.com/GumenchukAN/react_todo-app.git_
- Navigate to the project folder:
  - cd landing_bike-shop
- Install dependencies:
  - npm install_
- Run the project:
  - npm start_
